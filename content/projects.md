# Projects

## QGIS-ROS
A set of adapters to access ROS topics from within QGIS. Presentation: https://vimeo.com/293539252 Code: https://github.com/locusrobotics/qgis_ros


## Aiorospy
An `asyncio` wrapper for `rospy`. Code: https://github.com/locusrobotics/aiorospy

## The State Machine
A game where you try to visit every U.S. state once. Game: https://ablakey.github.io/state-machine/index.html

## Minibus
A Modbus server running on an ESP8266 Code: https://github.com/ablakey/minibus

## Leaflet Simple Graticule
A graticule, but for unprojected Cartesian spaces. Code: https://github.com/ablakey/Leaflet.SimpleGraticule

## Periodic Table of GitHub
Because naming repositories is hard so let's pick element names! Unfortunately this one doesn't work too well on mobile. Link: https://ablakey.github.io/periodic/index.html
